# Portfolio

My personal portfolio website — built from scratch with vanilla HTML, CSS, and JavaScript, featuring an interactive 3D background and smooth single-page navigation.

**Live site:** https://akumar962237.github.io/Portfolio/

## About

I'm a final-year Computer Science (AI & ML) student currently focused on full-stack web development, with Generative AI as my next area of study. This portfolio showcases my projects, skills, and ways to get in touch — built as a hands-on project to apply what I've learned along the way.

## Features

- Single-page app style navigation — Home, About, Skills, Projects, and Contact each render as their own view without page reloads
- Animated 3D background using Three.js (floating wireframe shapes with mouse parallax)
- Scroll-reveal animations and a subtle 3D tilt effect on hover
- Dark, code-editor inspired theme
- Skills and Projects sections with an owner-only edit mode (add/remove entries directly in the browser)
- Responsive layout for mobile and desktop

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, responsive design)
- Vanilla JavaScript
- [Three.js](https://threejs.org/) for the 3D background

## Running Locally

No build step or dependencies required.

1. Clone the repo:
   ```
   git clone https://github.com/akumar962237/Portfolio.git
   ```
2. Open `index.html` in a browser, or serve it locally (e.g. with the VS Code Live Server extension) for the best experience.
